DESCRIPTION
-----------
  "RedmondTux" is a game similar to SuperTux Team's "SuperTux v0.0.6"
  Run and jump through multiple worlds, fighting off enemies by jumping
  on them or bumping them from below. Grab power-ups and Linux distributions
  on the way.

  If you paid for this fangame, you were scammed. This fangame is entirely free, 
  as in freedom (you can do anything you want with it with only some limitations
  explained in the license) and price (you don't need to pay anything to play or
  modify this game)

STORY
-----
  Tux and Gown are having a picnic in Antarctica, when suddenly Gown is
  abducted! Tux must follow the path of clues across the worlds to find
  his friend and save her!

RUNNING THE GAME
----------------
  Just open the game using Godot. You can compile the game using Godot if you
  have the export templates for your Godot version installed.


TITLE SCREEN
------------
  You can use the mouse to click on any menu button.

  The Play Game button lets you play the game.
  
  The Credits button doesn't work yet but will let 
  you view the credits in the future.

  The Delete Save button lets you delete your save
  file. WARNING: IT WILL CLOSE THE GAME TOO!


  Status
  ------
    Your score is displayed at the upper left.
    Under your current score is the last highscore.

    The number of distros (CDs) you have collected is displayed below the
    score.

    This game has a health system, unlike SuperTux v0.0.6. It's displayed
    below the distros. You have a maximum of 3 health.

    There is no longer a timer or lives, spend as long as you want on a level
    and die as many times as you want in a level too.

SCORING
-------
  Doing things like defeating enemies and collecting distros will add score.

------------
